# The_Game
A model how [The Game](https://en.wikipedia.org/wiki/The_Game_%28mind_game%29) is spread

## Usage
Run with nimble
```bash
nimble run the_game
```
## Program Requirements
[Nim](https://nim-lang.org/) version 1.0.6 or greater
[Nimble](https://github.com/nim-lang/nimble)
